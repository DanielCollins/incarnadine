Incarnadine Engine

An experimental 3D rendering engine. You can obtain a copy of the source from the git repository at git://gitorious.org/incarnadine-engine/incarnadine-engine.git. Incarnadine is distributed under the zlib/libpng license, a copy of which is provided in the src directory. To Build and run Incarnadine you will also need to obtain a copy of the SDL library, available from http://www.libsdl.org/download-1.2.php.
